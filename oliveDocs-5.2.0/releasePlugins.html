    OLIVE 5.2.0 Release Plugins

    The following plugins have been tested and certified for compatibility and release with the OLIVE 5.2.0 software package.

    Speech

    • Speech Activity Detection (SAD)
      • sad-dnn-v7.0.2
    • Voice Type Discrimination (VTD)
      • vtd-dnn-v7.0.2

    Speaker

    • Speaker Identification (SID)
      • sid-dplda-v2.0.1
      • sid-embed-v6.0.1
    • Speaker Detection (SDD)
      • sdd-sbcEmbed-v2.0.3
    • Speaker Diarization (DIA)
      • dia-hybrid-v2.0.2
    • Speaker Highlighting (SHL)
      • shl-sbcEmbed-v1.0.2

    Language

    • Language Identification (LID)
      • lid-embedplda-v3.0.0
    • Language Detection (LDD)
      • ldd-embedplda-v1.0.0

    Gender

    • Gender Identification (GID)
      • gid-gb-v2.0.1
    • Gender Detection (GDD)
      • gdd-embedplda-v1.0.0

    Keyword

    • Query By Example (QBE)
      • qbe-tdnn-v5.0.1

    Transcription

    • Automatic Speech Recognition (ASR)
      • asr-dynapy-v3.0.0

    Translation

    Note that this TMT plugin currently has additional server and resource restrictions. Contact SRI for details.

    • Text Machine Translation (TMT)
      • tmt-statistical-v1.0.1

    Topic

    • Topic Detection (TPD)
      • tpd-dynapy-v5.0.0
      • tpd-fusion-v1.0.0

    Manipulation

    • Audio Redaction (RED)
      • red-transform-v1.0.0

    For further information on each individual plugin and their capabilities, please refer to the pages linked above. For more information about how to integrate with OLIVE and get started using these plugins, refer to the appropriate documentation linked from the landing page.
