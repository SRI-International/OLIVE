
This software allows you use Olive features from Java or Python that you write yourself.
It assumes you have already installed Olive and can run the oliveserver.py program.

If you are a developer, then read this file for instructions on setup for development:
README-Dev-Windows.txt

If you are a QA person, then ready this file for instructions on setup for testing:
src\main\python\olivepy\test\README-Testing.txt

If you are user of the Olive Python API, then you should not see this file at all, 
but if you do, please read this file:
README-Windows.txt
