    OLIVE 5.5.0 REST API Swagger Documentation

    Below is the Swagger for the OLIVE REST API; Note that this appears to be interactive, but messages submitted from this page won't properly reach the OLIVE Martini. However, if you currently have an OLIVE Martini container running, you can check out a live/interactive version of this Swagger documentation here:

    http://localhost:5005/swagger

    Or if the OLIVE Martini is running remotely:

    \<hostname>:5005/swagger
